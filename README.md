# Face_Mask_Detector
Implementation of face mask detector which can quickly be deployed on local camera based on MTCNN and resnet18.

## Requirement

```bash
pip install -r requirement.txt
```

## Runs

running main.py to open front camera and begin detection
```bash
python3 main.py
```
